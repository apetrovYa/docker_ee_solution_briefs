Download Splunk Monitoring Windows Container App
=================================================

From a web browser navigate to this URL: https://splunkbase.splunk.com/app/3858/. You will need to login using a Splunk account in order to download the App.
If you do not have a Splunk account, you can create an account for free.

Click on the **LOGIN TO DOWNLOAD** button.
